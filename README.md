### 使用技术

- IOC容器：Spring

- Web框架：SpringMVC

- ORM框架：Mybatis

- 安全框架：Shiro

- 数据源：C3P0

- 日志：log4j

- 前端框架：Bootstrap

- 集成开发环境：IntelliJ IDEA

- 项目构建工具：Maven

- 数据库：Mysql

- DK版本：jdk1.8

- Tomcat版本：Tomcat8
